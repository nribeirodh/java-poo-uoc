# Programación orientada a objetos en Java (UOC)

Este repositorio es donde estoy guardando las prácticas de **Programación Orientada
a Objetos en Java** de la UOC. No es ningún framework propio ni un proyecto súper
pulido, son ejercicios de la asignatura, pero organizados como proyectos Gradle
para que se vea el código y los tests de forma limpia.

La idea es que aquí se vea cómo trabajo en Java: diseño de clases, validaciones,
uso de colecciones, manejo de excepciones y pruebas con JUnit.


#PEC3. ejercicios de la PAC3  
  Modelos sencillos en Java para practicar POO:
  - Clases con atributos privados, getters/setters con validaciones y mensajes de error.
  - Uso de tipos como `LocalDate`, constantes de error, `toString`, etc.
  - Tests con JUnit para comprobar que las restricciones se cumplen.

#PEC4. Ejercicios de la PAC4  
  Ejemplos algo más complejos:
  - Jerarquías de clases, herencia y excepciones propias.
  - Uso de colecciones y enums.
  - Más batería de tests para asegurar el comportamiento.

#PR. Proyecto final de la asignatura  
  Mini aplicación para gestionar trenes/rutas/estaciones (nombre oficial de la práctica):
  - Proyecto Gradle con `src/main/java`, `src/test/java` y recursos.
  - Lógica separada en clases de modelo y controlador.
  - Lectura de datos desde ficheros y pruebas unitarias.
